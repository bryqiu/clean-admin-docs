## 什么是 Vue Clean Admin？

Vue Clean Admin 是一个中后台管理开发模板，它基于 Vue3 + Vite + TypeScript + Element
Plus 构建，旨在为开发者提供开箱即用，高效、可靠的中后台解决方案

项目特性介绍：

- **完善的工程：**
  现代化前端技术栈，全量 TypeScript 类型安全，完善的工程配置与开发规范，开箱即用的项目架构，易于扩展和维护
- **丰富的功能模块：**，内置细粒度权限设计、CRUD 快速开发、i18n 国际化、高质量组件封装、灵活主题系统、基于 ApiFox 的 Mock 数据
- **视觉设计：**，不算太丑的现代化 UI 设计，不算糟糕的视觉体验、用户体验
- **文档帮助：**，提供配套文档、[建设指南专栏](#文档及配套专栏)，助力开发者快速上手和深入学习

无论是个人开发者还是企业，Clean Admin 都能为你提供一个优秀的中后台解决方案

## 技术生态

开发者在选择中后台模板时，很需要考虑的是使用了什么技术来实现，这个技术我是否熟悉，是否能够快速上手

这里列出 Clean Admin 主要技术生态：

- Vue3 + Vite + TypeScript + Element Plus - 基础构建
- Tailwind CSS - 样式框架
- Pinia + pinia-plugin-persistedstate - 状态管理、持久化存储
- Vue Router - 路由管理
- Axios - HTTP 请求库
- Echarts - 图表库
- VueUse/Lodash-es - 组合式工具函数库、经典工具库(ESM)

你需要对以上技术有一定的了解，但不需要多精通

帮助你快速上手，是 Clean Admin 在做的事，你只需要专注于业务逻辑的实现

## 如何"食用"本文档

Clean Admin 的文档是基于 Astro + Starlight 构建的，Starlight 是在 Astro 框架之上构建的一个文档主题

> 想要帮助 Clean Admin 完善文档，不意味着你需要懂 Astro、Starlight，只需要懂 Markdown 语法即可

左侧目录栏主要分为五个部分：

- **入门指南**：想要了解并上手此项目的开发者，都应该先看看此部分
- **功能模块**：介绍 Clean Admin 中实现的功能模块，开发者会特别关注此部分
- **工程配置**：Clean Admin 项目的工程化配置，比如 Vite、类型系统、包管理工具
- **规范与约定**："没有规矩，不成方圆"，介绍项目正在严格遵守的规范准则
- **资源与支持**：项目相关的资源内容，或许有开发者想要找的内容
